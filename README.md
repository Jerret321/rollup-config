# rollup config

basic config of rollup for javascript development. Thanks @[haozi](https://github.com/haozime)

##  install 

```bash
npm install 
```

## use 

- change package.name with your package name
- **build**

```bash
npm run build
``` 
- **watch** 

```bash
npm run watch
```

- **test** 
```bash
npm run test
```
